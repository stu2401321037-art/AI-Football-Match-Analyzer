# AI Football Match Analyzer

AI Football Match Analyzer е модерно уеб приложение, което използва изкуствен интелект, за да предоставя задълбочен анализ и прогнози за футболни мачове. Приложението комбинира мощта на езикови модели с интуитивен интерфейс, за да предостави бързи и точни футболни инсайти.
g
## Основни функции

- Анализ на футболни мачове: Потребителят въвежда информация за мач, а системата генерира детайлен тактически и статистически анализ.
- Прогнозиране на резултати: AI моделът предвижда изхода от мача с вероятности, ключови фактори и очакван резултат.
- Автоматично извличане на: силни и слаби страни на отборите,ключови играчи,критични моменти и тн.
- Интерактивен уеб интерфейс: Лесна за използване система с табове за анализ и предикция.
- Реално време за обработка: Бързи отговори чрез интеграция с Google Gemini AI. 
- Адаптивен дизайн: Напълно оптимизиран интерфейс за работа на десктоп и мобилни устройства (Responsive Design).

## Използвана технология

- **Backend**: [FastAPI](https://fastapi.tiangolo.com/)
- **AI Model**: [Google Gemini API](https://ai.google.dev/)
- **Frontend**: HTML, CSS, JavaScript
- **Език**: Python 3.x

## Използвани библиотеки
- FastAPI: Основен уеб фреймуърк за изграждане на API ендпоинтите.
- google-genai: Официалната библиотека на Google за интеграция с Gemini моделите,използвана за заявките.
- uvicorn: ASGI сървър за стартиране на FastAPI приложението с висока производителност.
- python-dotenv: Използва се за сигурно зареждане на API ключовете от .env файл, избягвайки тяхното публикуване в публични хранилища.
- Pydantic: За валидация на данните и управление на настройките чрез Python типове.

## Изисквания
- Python 3.9 или по-нова версия
- Валиден API ключ от [Google AI Studio](https://aistudio.google.com/)

## Инсталация и стартиране
1. Клонирайте хранилището:
   ```bash
   git clone https://github.com/ВАШЕТО_ПОТРЕБИТЕЛСКО_ИМЕ/ВАШЕТО_РЕПО.git
   cd ВАШЕТО_РЕПО
   ```

3. Инсталирайте зависимостите:
   ```bash
   pip install -r requirements.txt
   ```

4. Създайте `.env` файл в папката на проекта:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

5. Стартирайте приложението:
   ```bash
   python main.py
   ```

6. Отворете приложението в браузър:
   ```
   http://127.0.0.1:5000
   ```
